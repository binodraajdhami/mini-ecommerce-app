## Mini Ecommerce Web App Developed with React JS
<p>Clone the repo app and follow the steps:</p>

## Step 1: cd mini-ecommerce-app
<pre>
    cd mini-ecommerce-app
</pre>

## Step 2: npm install
<pre>
    npm install
</pre>

## Step 3: npm start
<pre>
    npm start
</pre>
